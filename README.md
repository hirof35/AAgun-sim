# 対空砲シミュレーター (Anti-Aircraft Gun Simulator)

Node.js（サーバーサイド）とHTML5 Canvas（クライアントサイド）をWebSocketで連携させた、リアルタイムな対空砲撃シミュレーターです。
弾道の物理演算や近接信管（FLAK）、偏差射撃の概念を盛り込み、高射砲兵さながらの射撃体験をブラウザ上で再現します。

## 🚀 特徴・実装されている機能

* **リアルタイム通信**: Node.js側で毎秒60回（60fps）の物理演算を行い、WebSocket（`ws`）を通じてブラウザへ超低遅延で状態を同期。
* **弾道の放物線シミュレーション**: 発射された砲弾は上空へ向かうにつれ、重力（`GRAVITY`）の影響を受けてリアルな放物線を描きます。
* **近接信管（FLAK / VT信管）**: 敵機に直撃しなくても、一定の作動半径内（`PROXIMITY_RADIUS`）に到達すると自動で炸裂し、周囲にダメージ判定を持つ爆風を広げます。
* **偏差射撃ガイド（リードアシスト）**: 敵機の速度と距離、弾速から未来の位置を逆算し、画面上に薄い緑色の予測照準（レティクル）をリアルタイムに描画します。

## 🛠️ 技術スタック

* **サーバーサイド**: Node.js
* **ネットワーク**: WebSocket (`ws` ライブラリ)
* **クライアントサイド**: HTML5 Canvas, Vanilla JavaScript (フレームワーク不使用)

🎮 操作方法
マウス移動: 砲台の角度を調整（マウスカーソルの方向へ砲身が向きます）

左クリック: 対空砲を発射

💡 攻略のヒント（物理仕様の活かし方）
敵機は非常に高速で飛行しているため、現在の機体を直接狙っても弾は当たりません。
画面に表示される緑色の予測照準（リードマーカー）を狙うのが基本ですが、弾は重力で徐々に垂れ下がるため、「予測照準のさらに少し上」を狙うことで、放物線を描いた弾が上空の敵機に綺麗に吸い込まれるようになります。

🔧 パラメーターのチューニング
server.js の上部にある定数を書き換えることで、シミュレーターの難易度やリアリティを自由に調整できます。

BULLET_SPEED: 弾の初速。上げるほどレーザーのように真っ直ぐ飛び、当てるのが容易になります。

GRAVITY: 弾にかかる重力。大きくするほど弾道が垂れ下がり、高角での射撃が難しくなります。

PROXIMITY_RADIUS: 近接信管の感度。大きくするほどアバウトな狙いでも炸裂するようになります。

JavaScript
// server.js 内の調整可能パラメーター
const GRAVITY = 0.12;          
const PROXIMITY_RADIUS = 40;  
const BLAST_RADIUS = 60;      
const BULLET_SPEED = 18;      
(※ BULLET_SPEED を変更した際は、index.html 内の bulletSpeed 定数も同じ数値に同期させてください)
